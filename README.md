# Password Manager

## Description
The Password Manager is a secure desktop application developed in Java using the Swing framework for its graphical user interface.It allows users
to safely store, manage, and retrieve passwords for multiple websites and applications.Passwords are encrypted using AES encryption to ensure that
even if the storage file is accessed by unauthorized users, the data remains protected.This application also includes a login system, requiring users
to authenticate before accessing stored passwords, adding an extra layer of security.It is designed to be simple and user-friendly, making it easy for
anyone to manage their sensitive information without worrying about security risks.The app also supports features like viewing all stored passwords in
a table, toggling password visibility, and avoiding duplicate entries to keep the data organized.  

Overall, this Password Manager is a lightweight yet powerful tool for securely managing your personal and professional passwords in one place.

## Features
- ✅ User login authentication
- ✅ Add, retrieve, and store passwords
- ✅ AES encryption for password security
- ✅ View all stored passwords in a table
- ✅ Eye icon to toggle password visibility
- ✅ Simple and user-friendly GUI
- ✅ Avoids duplicate entries in the password file
